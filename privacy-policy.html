我们，Email-check-extensio，作为Email-check-extensio浏览器扩展（“浏览器扩展”）的提供商， 非常重视数据保护问题，并希望确保您作为我们服务用户的隐私受到保护。

1. 个人资料的使用

我们可能仅将您的个人信息用于以下一个或多个目的：

为您提供支持。例如，如果您通过电子邮件留下您的个人信息，我们可能会与您联系，以帮助您解决问题或回答您的问题。

推广我们的服务。例如，如果我们认为您可能会从使用我们提供的其他产品或服务中受益，或者如果我们认为有关当前服务更改的信息与您相关，我们可能会与您联系以告知您相关情况。

2. 处理和使用您的数据

我们收集、处理和使用以下个人数据：

2.1 联系我们

如果您出于联系我们的目的或与联系我们有关的目的向我们提供个人数据，则这些数据将仅由我们存储，直到沟通、客户联系和项目规划/实施目的需要。一旦我们不再需要个人数据用于这些目的，它们将立即删除。

2.2 使用网站

我们不使用 cookie，我们的托管服务提供商仅以缩短形式记录 IP 地址。为了获取有关我们网站流量的基本信息，我们使用与隐私相关的Google Analytics替代 plausible.io，该替代绝对匿名地处理所有数据，而无需持久标识符（cookie）。数据既不会跨站点或设备进行跟踪，也不会用于任何其他目的。

2.3 下载Email-check-extension扩展

2.3.1 Chrome扩展程序

Chrome 扩展程序托管在 Chrome 网上应用店中。请参阅 www.google.com/policies/privacy，了解在下载扩展程序或在定期更新期间收集哪些数据的信息。

2.3.2 Firefox 附加组件

Firefox 附加组件托管在 addons.mozilla.org （AMO） 上。请参阅 www.mozilla.org/en-US/privacy，了解在下载扩展程序或在定期更新期间收集哪些数据的信息。

2.3.3 边缘扩展

边缘扩展托管在 Microsoft Edge 加载项上。请参阅 https://privacy.microsoft.com/en-us/，了解在下载扩展程序或在定期更新期间收集哪些数据的信息。

2.5 使用浏览器扩展

Email-check-extensio仅将数据存储在浏览器的本地存储中，其他应用程序无法访问该存储。

Email-check-extensio将在设置过程中要求用户同意与我们共享汇总和匿名的分析信息。我们使用 Clean Insights 衡量平台来帮助更好地了解应用使用模式。数据是以保护隐私的方式捕获的：我们不使用跟踪 cookie，我们不存储 IP 地址，我们不捕获个人信息，我们不跨站点跟踪用户。我们使用 Matomo 来收集有关您使用浏览器扩展程序的信息。

3. 同意声明

在接受本数据保护政策时，您可撤销地同意我们收集、处理和使用您直接或间接提供给我们的个人数据。

4. 个人数据的共享

除非您事先明确同意，或者法律规定或法律允许，否则我们不会与第三方共享您的个人数据。我们不会将您的数据出售给第三方，也不会以任何其他方式进行营销。

但是，只有在以下情况下才能共享个人数据：

  • 如果政府机构和当局提出要求，则仅遵守强制性国家法律。到目前为止，这从未发生过。
  • 用于向代表和顾问提供信息。其中包括工程师、律师和会计师，他们帮助我们遵守法律、会计或安全要求。我们的员工和合作伙伴有义务保密并遵守数据保护法规。
  • 为了在出售、合并、合并或收购的情况下转移您的信息，任何收购方都将遵守我们在本隐私政策下的义务，包括您编辑和删除您的个人数据的权利。我们将通过向您发送电子邮件来通知您更改，或者如果我们没有您的联系数据，则通过在我们的网站上发布通知来通知您，以便您可以根据需要选择退出。

5. 信息、更正、删除

您可以随时免费获得有关我们存储的有关您的个人数据的信息。如果您的数据不正确或被我们错误地存储，我们很乐意更正、阻止或删除此类数据。如果您的数据有任何更改，请立即通知我们。
